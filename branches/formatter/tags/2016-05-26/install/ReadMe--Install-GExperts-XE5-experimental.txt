Steps to install:
0- Quit Delphi XE5
1- Download the GExperts XE5 experimental from http://....../
2- Recursively unpack that download
3- Copy the batch file Install-GExperts-XE5-experimental--run-as-administrator-from-ExpertManager.exe-directory.bat into that directory
4- From that directory, run the copied Install-GExperts-XE5-experimental--run-as-administrator-from-ExpertManager.exe-directory.bat
5- Run Delphi XE5
6- Check if GExperts is installed

Note:
- this batch file copies regularexpert\GExpertsRSXE5.dll
- copy editorexpert\GExpertsRSXE5.dll manually after installation if you prefer that one.
